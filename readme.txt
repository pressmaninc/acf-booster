== Description ==
"ACF Booster" is a plugin which boosts up  the functionality of Advanced Custom Fields.
We will add more features soon.

== Features ==
* **NG Word Checker**
Sets prohibited words in the specified text field.

* **Character Counter**
Adds a counter to text fields with character limits.

== Requirements ==
**This plugin requires ACF Pro **
[ACF Pro](https://www.advancedcustomfields.com/pro/)

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the plugin package to the plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

Yes, supports Classic Editor and Gutenberg.
